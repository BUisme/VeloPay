โครงสร้างไฟล์ (แยก HTML/CSS/JS) สำหรับ GitHub Pages

- index.html
- assets/
    - css/
        - styles.css
    - js/
        - app.js

วิธีใช้งาน:
1) แตกไฟล์ zip
2) อัปโหลดทั้งโฟลเดอร์/ไฟล์ทั้งหมดเข้า repo (ไว้ที่ root)
3) GitHub > Settings > Pages > Branch: main, Folder: /(root)
4) เปิดลิงก์ GitHub Pages ได้เลย

หมายเหตุ:
- ถ้า URL มี #z=... หรือ #s=... จะเป็นโหมดดูอย่างเดียว
- มีปุ่ม "กลับไปโหมดจัดทำสัญญา" บนหน้า Report
